# Crop-Recommendation-System
The Crop Recommendation System is an intelligent solution designed to help farmers choose the most suitable crop based on environmental and soil conditions. Agriculture depends on factors like temperature, rainfall, humidity, and soil nutrients to ensure better crop growth and higher yield.

# Crop Recommendation System

## Project Overview  
This project simulates a smart system that recommends the best crop based on input conditions such as temperature, humidity, rainfall, and soil pH. It helps farmers make better decisions by suggesting crops that are more likely to grow successfully in given conditions.

## Features  
1. Prediction of suitable crop based on input parameters  
2. Uses machine learning for decision making  
3. Simple and easy-to-use system  
4. Helps improve crop yield and reduce risk  

## Technologies Used  
Python 3  
Pandas and NumPy for data handling  
Scikit-learn for machine learning  

## How to Run  
Ensure Python 3 is installed.  
Run the script: python main.py  
Enter the required values and get the recommended crop.
